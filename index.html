Projetos

Delivery - Pizzaria

O projeto consiste em um site para uma pizzaria fictícia, e possui funcionamento responsivo em diferentes dispositivos. A página conta com um carrinho de compras totalmente funcional, permitindo que o usuário adicione e remova pizzas no carrinho. Além disso, o site também aplica descontos diretamente no carrinho de compras. Para tornar o processo de compra ainda mais fácil, o site utiliza objetos para armazenar informações sobre as pizzas, tornando a navegação mais rápida e eficiente.

Tecnologias usadas no projeto: HTML, CSS, JavaScript Vanilla.
